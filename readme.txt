== Description ==

This plugin adds the metisMenu jQuery plugin to chosen WordPress menus.



== Installation ==

This section describes how to install the plugin:

1. Upload the `metismenu` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Options are in Settings => metisMenu


== Frequently Asked Questions ==

= How do I find the Menu CSS ID =

Highlight the Menu in the browser, for example in Chrome, right click and inspect element - then look for the <code><ul id="menu-name-here">.....</ul></code>

When adding the Menu in the metisMenu settings make sure to add the CSS ID symbol eg. #menu-name-here or .menu-classname


== Usage ==

The settings are found via the dashboard Settings > metisMenu

- Add in the Menu CSS ID to apply the metisMenu to, comma separate multiple menus.
- Each menu transformed into a metisMenu will get an additional CSS Menu class called 'metismenu'
- Option to enable 'Toggle' effect which can close menu item when another is clicked
- Option to disable the metisMenu default CSS
